Overview

This page provides access to a compressed SQLite snapshot of the database used by the web application. The dataset is intended for offline analysis, reproducibility, and exploratory research using standard SQLite-compatible tools (e.g., sqlite3, DB Browser for SQLite, or Python libraries).

Format: SQLite (compressed)
Primary table: vital_signs_metrics
Database structure

The dataset exposes a single main table, vital_signs_metrics, which stores aggregated metrics computed over Wikipedia editing activity.

Table: vital_signs_metrics

  • langcode (TEXT)
  • year_year_month (TEXT)
  • year_month (TEXT)
  • topic (TEXT)
  • m1 (TEXT)
  • m1_calculation (TEXT)
  • m1_value (TEXT)
  • m2 (TEXT)
  • m2_calculation (TEXT)
  • m2_value (TEXT)
  • m1_count (FLOAT)
  • m2_count (FLOAT)
Tip: you can query this table directly with sqlite3 or load it into Python using pandas.read_sql_query after decompressing the file.
